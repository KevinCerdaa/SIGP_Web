Acerca de Nosotros

¿Qué es SIGP?

SIGP (Sistema de Identificación de Grupos Pandilleriles) es una plataforma tecnológica diseñada para apoyar a las autoridades y organizaciones en la gestión, consulta y análisis de información relacionada con grupos pandilleriles y actividades delictivas.

Nuestro sistema proporciona herramientas avanzadas para visualizar ubicaciones en mapas interactivos, realizar consultas complejas, generar reportes detallados y mantener un registro organizado de pandillas, sus integrantes y eventos relacionados.

Nuestra Misión

Proporcionar una herramienta eficiente, segura y confiable que facilite la identificación, seguimiento y análisis de grupos pandilleriles, contribuyendo así a la seguridad pública y al trabajo de las autoridades competentes.

Buscamos mejorar la capacidad de respuesta y toma de decisiones mediante el acceso rápido y organizado a información relevante, siempre respetando los más altos estándares de privacidad y seguridad de datos.

Características Principales

Mapas Interactivos

Visualización geográfica de ubicaciones con marcadores codificados por nivel de peligrosidad y mapas de calor.

Consultas Avanzadas

Sistema de búsqueda flexible por eventos, pandillas, integrantes y consultas generales con múltiples filtros.

Generación de Reportes

Exportación de consultas a formato PDF con información detallada y estadísticas relevantes.

Seguridad y Privacidad

Sistema de autenticación robusto y control de acceso basado en roles para proteger la información sensible.

Tecnologías Utilizadas

SIGP está desarrollado utilizando tecnologías modernas y confiables para garantizar un rendimiento óptimo y una experiencia de usuario fluida:

  • Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
  • Backend: Django REST Framework, Python
  • Base de Datos: MySQL
  • Mapas: Google Maps API con visualización de datos
  • Seguridad: Autenticación por tokens, CORS configurado
