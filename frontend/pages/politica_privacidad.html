Política de Privacidad

Última actualización: Enero 2025

En SIGP (Sistema de Identificación de Grupos Pandilleriles), nos comprometemos a proteger la privacidad y seguridad de la información que manejamos. Esta Política de Privacidad describe cómo recopilamos, utilizamos, almacenamos y protegemos los datos personales y la información sensible en nuestro sistema.

1. Información que Recopilamos

El sistema SIGP recopila y almacena la siguiente información:

  • Datos de usuarios: Información de autenticación, roles y permisos de acceso.
  • Datos de pandillas: Nombres, líderes, descripciones, grados de peligrosidad y zonas de operación.
  • Datos de integrantes: Nombres completos, alias, fechas de nacimiento, direcciones y relaciones con pandillas.
  • Datos de direcciones: Ubicaciones geográficas (coordenadas), calles, números, colonias y zonas.
  • Datos de eventos: Fechas, horas, tipos de eventos, delitos o faltas asociadas, y descripciones.
  • Datos de delitos y faltas: Catálogos de delitos y faltas administrativas registradas.

2. Uso de la Información

La información recopilada se utiliza exclusivamente para los siguientes propósitos:

  • Facilitar la identificación y seguimiento de grupos pandilleriles.
  • Proporcionar herramientas de consulta y análisis para las autoridades competentes.
  • Generar reportes y visualizaciones geográficas para apoyo en la toma de decisiones.
  • Mantener un registro organizado y actualizado de información relevante para la seguridad pública.
  • Gestionar el acceso y los permisos de los usuarios del sistema.

3. Seguridad de los Datos

Implementamos medidas de seguridad técnicas y organizativas para proteger la información:

  • Autenticación: Sistema de autenticación por tokens para garantizar que solo usuarios autorizados accedan al sistema.
  • Control de acceso: Sistema de roles y permisos que restringe el acceso según las responsabilidades de cada usuario.
  • Encriptación: Las contraseñas se almacenan de forma segura utilizando algoritmos de hash.
  • Conexiones seguras: El sistema utiliza protocolos seguros para la transmisión de datos.
  • Base de datos protegida: La base de datos está protegida y solo es accesible por personal autorizado.
  • Auditoría: Se registran las actividades de los usuarios para mantener un historial de acceso y modificaciones.

4. Acceso y Control de Datos

El acceso a la información está estrictamente controlado:

  • Solo usuarios autenticados con credenciales válidas pueden acceder al sistema.
  • Los permisos de acceso se asignan según el rol y las responsabilidades de cada usuario.
  • Los administradores tienen control total sobre la gestión de usuarios y datos.
  • Los editores pueden modificar y agregar información según sus permisos.
  • Los usuarios con permisos limitados solo pueden consultar información.

5. Compartir Información

La información almacenada en SIGP es confidencial y no se comparte con terceros, excepto en los siguientes casos:

  • Cuando sea requerido por ley o por orden judicial.
  • Cuando sea necesario para proteger la seguridad pública o prevenir actividades delictivas.
  • Con autoridades competentes que tengan autorización legal para acceder a la información.

No vendemos, alquilamos ni compartimos información con fines comerciales.

6. Retención de Datos

Los datos se conservan mientras sean necesarios para los fines para los que fueron recopilados, o según lo requiera la ley. Los datos obsoletos o que ya no son necesarios pueden ser eliminados siguiendo los protocolos establecidos y las regulaciones aplicables.

7. Derechos de los Usuarios

Los usuarios del sistema tienen derecho a:

  • Acceder a su información de cuenta personal.
  • Solicitar correcciones de información incorrecta.
  • Reportar problemas de seguridad o acceso no autorizado.
  • Solicitar información sobre cómo se utiliza y protege su información.

Para ejercer estos derechos, contacta al administrador del sistema o envía un correo a contacto@sigp.com.mx.

8. Cookies y Tecnologías de Seguimiento

SIGP utiliza tokens de autenticación almacenados localmente para mantener las sesiones de usuario. Estos tokens son necesarios para el funcionamiento del sistema y se eliminan al cerrar sesión.

El sistema también utiliza Google Maps API para la visualización de mapas. El uso de esta API está sujeto a las políticas de privacidad de Google.

9. Cambios a esta Política

Nos reservamos el derecho de actualizar esta Política de Privacidad en cualquier momento. Los cambios significativos serán notificados a los usuarios a través del sistema o por correo electrónico. La fecha de la última actualización se indica al inicio de este documento.
